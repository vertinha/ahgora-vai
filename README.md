## Desenvolvimento

1. Você vai precisar do [yarn](https://yarnpkg.com/lang/en/docs/install/) e um Node.js >= 6.
2. Execute `yarn`.
3. Execute `npm run start`.
4. Carregue a extensão no Chrome seguindo:
   1. Acesse `chrome://extensions/`
   2. Ligue o `Modo do desenvolvedor`
   3. Clique em `Carregar sem compactação`
   4. Selecione a pasta `build`.
